# Min-studieplanerare
Här är en app skapad av mig, Natalie Lundin, där du kan planera upp dina studier efter uppgift och deadline. Du har färgordnade uppgifter, nedräkning till din stora dag och studieverktyg. Nu kan du också infoga direktlänkar som gör det lätt för dig att komma åt dina uppgifter direkt ifrån appen.
